How it works

Simple by design

AEP is a pure observability layer. Your agents emit structured events; AEP captures, validates, and visualises them.

01

Emit events

POST a JSON event envelope from your agent — tasks, tool calls, memory ops, handoffs, errors.

02

Ingest & validate

The server validates against JSON Schema, deduplicates, and stores — with optional HMAC auth.

03

Trace & debug

The live dashboard reconstructs session timelines, causation DAGs, and multi-agent trees.

Features

Everything you need to observe agents

A complete stack — protocol, ingest, storage, visualisation, and CLI.

📋

Standard event schema

12 core event types across 5 categories. Auto-ID, timestamps, and payload validation built in.

⚡

Live dashboard

Real-time session replay, causation DAG visualisation, and multi-agent workflow trees.

🔌

Production-ready ingest

API key auth, rate limiting, deduplication, HMAC signing, and Prometheus metrics export.

🛠️

CLI tooling

Emit events, query sessions, export data, and validate payloads — all from the terminal.

Quick start

Up in 60 seconds

Requires Node.js 20+. No external database or cloud account needed.

# 1. Install dependencies
npm install
# 2. Start the ingest server
npm run ingest
# 3. Emit a sample event
npm run emit:example
# 4. Open the dashboard (default port: 8787)
open http://localhost:<PORT>/dashboard
Try a built-in demo
npm run demo:support
